⚡ ocrplus

焊机参数 OCR 智能识别系统 — Jetson Orin Nano 边端部署

Jetson Orin Nano CUDA 12.6 FastAPI C++ PaddleOCR

🎯 应用场景

本系统专为工业焊机参数实时识别设计。在焊接作业中，焊机表头的电流、电压、 焊接速度等参数需要实时监控和记录。通过在焊机表头安装 ESP32 + 摄像头模组， 定时拍摄表头屏幕，将图像通过 WiFi 回传至 Jetson Orin Nano 上运行的 OCR API 服务， 自动识别并结构化输出焊机参数。

🏁 快速开始

1. 克隆仓库

git clone https://github.com/tydfgt/ocrapi.git
cd ocrapi

2. 安装依赖

source venv/bin/activate
pip install -r requirements.txt

3. 启动服务

bash start.sh
# 访问 http://[设备IP]:8899

4. ESP32 端调用示例

# ESP32 拍照后通过 HTTP POST 上传
curl -X POST http://192.168.3.110:8899/ocr \
  -F "file=@weld_params.jpg"

🔌 API 接口

方法路径说明
POST/ocr上传图片文件，返回 OCR 识别结果（JSON）
POST/ocr_base64Base64 编码图片识别
GET/status实时硬件状态（GPU/CPU/RAM/温度/功耗/风扇）
GET/health健康检查
GET/Web 管理面板（含硬件监控）
