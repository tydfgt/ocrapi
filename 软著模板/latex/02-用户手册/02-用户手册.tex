\section{安装与卸载}

\subsection{软件安装}

\subsubsection{部署步骤（Web端）}
\begin{enumerate}
    \item 确保服务器满足运行环境要求
    \item 安装所需依赖环境（如 JDK、数据库等）
    \item 将程序包解压至目标目录
    \item 配置数据库连接信息（修改 \texttt{application.yml} 配置文件）
    \item 执行数据库初始化脚本
    \item 启动服务：\texttt{java -jar xxx.jar}
    \item 访问地址：\texttt{http://localhost:8080}
\end{enumerate}

\subsubsection{安装步骤（客户端）}
\begin{enumerate}
    \item 双击安装包 \texttt{Setup.exe}
    \item 阅读并同意许可协议
    \item 选择安装目录
    \item 点击“安装”按钮
    \item 等待安装完成，点击“完成”
\end{enumerate}

\subsection{软件卸载}
\begin{enumerate}
    \item 打开“控制面板” → “程序和功能”
    \item 找到【软件名称】
    \item 右键选择“卸载”，按提示完成卸载
\end{enumerate}
