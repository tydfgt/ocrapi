\vspace{0.5cm}

\textbf{摘要}：

本软件是一套运行于 NVIDIA Jetson Orin Nano 边缘计算平台上的智能 OCR 文字识别系统。系统利用 PaddlePaddle 深度学习框架的 C++ 推理引擎与 PP-OCRv5 模型，在 ARM 架构的边缘设备上实现高效、低延迟的图片文字检测与识别功能。软件提供 RESTful API 服务接口和 Web 可视化前端，支持 GPU 加速推理和实时硬件状态监控。

系统采用四层架构设计：用户交互层（HTML5 前端）、API 服务层（FastAPI）、C++ 推理引擎层（Paddle Inference + OpenCV CUDA）和硬件平台层（Jetson Orin Nano）。推理引擎基于 CUDA 12.6 和 cuDNN 9.3 进行 GPU 加速，预热后每张图片推理耗时约 5.5 秒。
