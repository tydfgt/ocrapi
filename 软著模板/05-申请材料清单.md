# 软著申请材料清单

---

## 📋 必需材料

| 序号 | 材料名称 | 要求 | 状态 |
|------|----------|------|------|
| 1 | **软件著作权登记申请表** | 在线填写并打印，加盖公章（企业）或签字（个人） | ☐ |
| 2 | **软件说明书（用户手册）** | 图文并茂，包含软件界面截图，PDF 格式 | ☐ |
| 3 | **源代码文档** | 前后各 30 页，每页 ≥ 50 行，PDF 格式 | ☐ |
| 4 | **营业执照副本**（企业）/ **身份证复印件**（个人） | 加盖公章（企业）/ 签字（个人） | ☐ |

## 🔍 审核要点自查

### 一致性检查

- [ ] 所有材料中**软件名称**完全一致（含大小写、标点、版本号）
- [ ] 所有材料中**著作权人名称**一致
- [ ] 开发完成日期在各文档中一致
- [ ] 版本号格式统一（如 V1.0、V1.0.0）

### 源代码检查

- [ ] 源代码共 60 页
- [ ] 前 30 页包含程序核心逻辑代码
- [ ] 后 30 页为程序末尾代码
- [ ] 每页不少于 50 行有效代码
- [ ] 无第三方库、自动生成代码
- [ ] 页眉标注软件名称 + 版本号
- [ ] 页脚标注页码

### 说明书检查

- [ ] 包含软件运行截图（建议 8-10 张以上）
- [ ] 每个核心功能均有操作说明
- [ ] 包含运行环境说明
- [ ] 包含安装/部署步骤
- [ ] 总页数建议 15-30 页

### 申请表检查

- [ ] 所有必填项已填写完整
- [ ] 软件分类号正确
- [ ] 权利范围已勾选
- [ ] 软件功能简述完整清晰（200-500 字）

---

## 📮 提交流程

1. 登录 [中国版权保护中心](http://www.ccopyright.com.cn) 官网
2. 在线填写软件著作权登记申请表
3. 打印申请表并签字/盖章
4. 将所有纸质材料邮寄至版权保护中心
5. 等待受理通知（约 1-2 周）
6. 等待审查结果（约 2-4 个月）
7. 领取著作权登记证书
