### 1.2 提取脚本

以下 Python 脚本自动提取所有原创源代码文件，过滤第三方库和自动生成代码：

```python
#!/usr/bin/env python3
"""
源代码提取脚本 — 用于软著申请
输出: 源代码文档.txt（拼接所有源码）
"""
import os
from pathlib import Path

PROJECT_ROOT = Path("/home/ysdhanji/ocrcplus")
OUTPUT_FILE = PROJECT_ROOT / "软著模板" / "源代码文档.txt"

# 需提取的文件（原创代码）
INCLUDE_FILES = [
    "server.py",
    "start.sh",
    "stress_test.py",
    "static/index.html",
    "requirements.txt",
]

INCLUDE_DIRS = [
    "PaddleOCR/deploy/cpp_infer/src",
    "PaddleOCR/deploy/cpp_infer/cli.cc",
    "PaddleOCR/deploy/cpp_infer/CMakeLists.txt",
]

# 排除模式（第三方/自动生成）
EXCLUDE_PATTERNS = [
    "build/", "build_trt/", "third_party/", ".git/",
    "venv/", "models/", "paddle_inference_install_dir/",
    "__pycache__/", "*.pyc", "*.o", "*.so", "*.a",
    "Paddle/build/", "backup/",
]


def should_exclude(path: str) -> bool:
    for pat in EXCLUDE_PATTERNS:
        if pat.endswith("/") and pat[:-1] in path.split("/"):
            return True
        if pat.startswith("*") and path.endswith(pat[1:]):
            return True
        if pat in path:
            return True
    return False


def collect_files() -> list:
    files = []
    for f in INCLUDE_FILES:
        p = PROJECT_ROOT / f
        if p.exists() and not should_exclude(str(p)):
            files.append(p)

    for d in INCLUDE_DIRS:
        p = PROJECT_ROOT / d
        if p.is_file() and not should_exclude(str(p)):
            files.append(p)
        elif p.is_dir():
            for root, dirs, fnames in os.walk(p):
                dirs[:] = [d for d in dirs if not should_exclude(d)]
                for fn in sorted(fnames):
                    fp = Path(root) / fn
                    if not should_exclude(str(fp)):
                        files.append(fp)

    return sorted(set(files))


def generate():
    files = collect_files()
    with open(OUTPUT_FILE, "w", encoding="utf-8") as out:
        for fp in files:
            rel = fp.relative_to(PROJECT_ROOT)
            out.write(f"\n// ===== {rel} =====\n\n")
            try:
                content = fp.read_text(encoding="utf-8")
                out.write(content)
            except Exception as e:
                out.write(f"// Error reading: {e}\n")
            out.write("\n")

    print(f"✅ 已提取 {len(files)} 个文件 → {OUTPUT_FILE}")
    # 统计行数
    total_lines = sum(1 for _ in open(OUTPUT_FILE, encoding="utf-8"))
    print(f"   总行数: {total_lines}")


if __name__ == "__main__":
    generate()
```

## 二、源代码提交规范

### 2.1 总体要求

- 提交的源程序需为**完整代码**，具有原始性
- 总共提交 **60 页** 源代码（前 30 页 + 后 30 页）
- 每页代码不少于 **50 行**
- 页眉标注软件名称及版本号："PaddleOCR 智能文字识别系统 V1.0"
- 页脚标注页码："第 X 页 / 共 60 页"

### 2.2 格式要求

| 项目 | 要求 |
|------|------|
| 纸张大小 | A4 |
| 页边距 | 上下 2.54cm，左右 3.18cm |
| 字体 | 等宽字体（Consolas / Courier New / Source Code Pro） |
| 字号 | 小五号（9pt） |
| 行间距 | 单倍行距 |
| 页眉 | 软件全称 + 版本号（居中） |
| 页脚 | 页码（居中） |

### 2.3 内容组织

- **前 30 页**：从程序开头选取——API 服务核心代码、C++ 推理引擎初始化、命令行入口
- **后 30 页**：从程序末尾选取——工具函数、日志系统、CMake 构建配置、HTML 前端
- 每段文件开头添加注释标注文件名，如：`// ===== src/server.py =====`
- 同一文件内容连续排列，不同文件之间用空行和注释分隔

### 2.4 不纳入源代码的内容

- 第三方开源库代码（OpenCV、Paddle Inference、FastAPI 等）
- 自动生成代码（CMake 生成的 Makefile、Python .pyc 等）
- node_modules / vendor 等依赖目录
- 编译产物（.o、.so、.exe 等）
- 配置文件中的敏感信息（可脱敏后保留）
