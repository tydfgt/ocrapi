# 软件著作权申请模板

本文件夹包含软件著作权（软著）申请所需的各类文档模板，请根据实际项目情况填写。

## 文件说明

| 文件名 | 用途 |
|--------|------|
| `01-软件著作权登记申请表.md` | 向中国版权保护中心提交的申请信息 |
| `02-软件说明书（用户手册）.md` | 软件使用说明，供审查人员了解软件功能 |
| `03-源代码文档说明.md` | 源代码提交规范与格式要求 |
| `04-软件设计说明书.md` | 软件技术方案与设计文档（可选） |
| `05-申请材料清单.md` | 材料准备 check list |

## 使用步骤

1. 按顺序填写各模板文件
2. 源代码需单独整理（前后各 30 页，共 60 页）
3. 所有文档最终需导出为 PDF 格式提交
4. 申请表需在线填写并打印（中国版权保护中心官网）

## 注意事项

- 源代码每页不少于 50 行（建议 55 行左右）
- 用户手册需图文并茂，包含软件界面截图
- 所有材料中软件名称、版本号必须保持一致
